# Weather App

A simple weather applicatiob built with [React](https://reactjs.org/) framework.

## Installation

1. Clone the repo
2. Install [Node.js](https://nodejs.org) packages
3. Enjoy!

## Usage

###### Run the App

```node
npm run start
```

###### Test the App

```node
npm run test
```

###### Build the App

```node
npm run build
```
